# install

**Termux**

```
pkg install git
pkg install php
git clone https://github.com/Ranginang67/kinWeb
cd kinWeb
php kinweb.php
```

**Linux**

```
apt-get install git
apt-get install php
apt-get install php-curl
git clone https://github.com/Ranginang67/kinWeb
cd kinWeb
php kinweb.php
```
